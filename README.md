# 👨‍🍳 SmartChef AI – Intelligent Cooking & Nutrition Assistant

SmartChef AI is a full-stack AI-powered web application that helps users cook smarter and plan meals efficiently. It leverages LLaMA-3.1-8b-instant model to generate personalized recipes, nutritional insights, and structured meal plans through a modern, responsive interface.


## ✨ Features

### 🍳 Intelligent Recipe Generator

Generate complete recipes based on available ingredients, including:

* Step-by-step cooking instructions
* Cooking time
* Calories and macro-nutrients
* Nutritional benefits
* Health suitability guidance

### 📅 Goal-Based Meal Planner

Create structured weekly or monthly meal plans based on:

* Weight loss
* Muscle gain
* Diabetic-friendly diet
* General healthy eating

Includes automatic grocery list generation.

### 🤖 AI Cooking Chatbot

An always-available chatbot to:

* Answer cooking-related questions
* Suggest ingredient substitutions
* Provide diet recommendations
* Recommend budget-friendly meals

### 🎨 Modern User Interface

* Light / Dark theme toggle
* Smooth animations
* Glassmorphism-inspired design
* Fully responsive layout
